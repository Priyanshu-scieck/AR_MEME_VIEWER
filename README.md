An Augmented Reality application built using **Unity** and **Vuforia** that displays memes on detecting an image target.  
Users can scan an image, view memes in AR, and cycle through memes using an update button.

## Features
- Image Target based AR using Vuforia
- Fetches memes dynamically from GitHub
- Sequential meme switching
- Clean menu-based UI (Scan Target / Exit)
- Android support

## Tech Stack
- Unity 6
- Vuforia Engine
- C#
- GitHub (for meme hosting)

## How to Run
1. Clone the repository
2. Open the project in Unity Hub
3. Add your Vuforia License Key
4. Build and run on Android
